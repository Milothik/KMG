# Kawaii Mixer Game v1.0.0

Primera release jugable de Kawaii Mixer Game para pruebas privadas en Android.

Kawaii Mixer Game es un juego mobile-first de mezcla de perfumes: recoge gotas en el memory, combina Massnotas en el panel de juego y crea acordes olfativos para responder al pedido de cada clienta.

## Descargar

Para Android, descarga e instala:

`kawaii-mixer-game-release.apk`

Si Android muestra "Aplicación no instalada" al actualizar desde una versión anterior, desinstala la APK antigua y vuelve a instalar la APK de release.

## Incluye

- Pantalla inicial con acceso a `Jugar`, Perfupedia, Mixer de Cautivos y Elaboraciones.
- Sistema de pedidos con clientas, tono aromático y frases de encargo.
- Memory táctil para conseguir gotas y materia prima.
- Panel de mezcla por fases: `Fondo`, `Corazón` y `Salida`.
- Lectura de acorde actual con perfiles como fresco, floral, dulce, sensual y verde.
- Massnotas, solvente, sistema Maggic y cautivos consumibles.
- Progreso persistente en el dispositivo.
- Controles separados de música y efectos.
- Icono de aplicación incluido en Android.

## Correcciones importantes de Android

- La APK empaqueta correctamente `index.html` dentro de `android_asset/www`.
- Los datos de niveles se incluyen en la build para que el botón `Jugar` cargue la partida.
- Los assets visuales de interfaz y paneles están integrados en la app instalada.
- Se ha añadido icono de launcher para que la app no aparezca con el icono generico.

## iPhone e iPad

Esta release también deja preparado el flujo de iOS en Codemagic para generar una IPA sin firma comercial. Esa IPA puede servir para pruebas con Sideloadly, pero para instalarla oficialmente, usar TestFlight o distribuirla por App Store hace falta una cuenta de Apple Developer con certificados y perfiles de firma.

## Estado

Esta versión es una pre-release de pruebas. El objetivo es validar la experiencia principal en móvil, revisar balance de tutorial/niveles y confirmar que la instalación Android funciona correctamente fuera del servidor local.
